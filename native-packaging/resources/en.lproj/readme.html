README for GNAT-CE-2021

This is GNAT CE 2021 built on Mac OS X El Capitan (10.11, Darwin 15), with Xcode 8.2.1.

Also runs on macOS versions up to Big Sur (11, Darwin 20).

Unlike previous GCC packages for macOS, this package installs at a fixed location: /opt/gnat-ce-2021.

Compilers included: Ada, C, C++.

Tools included (all under GPL v3), from AdaCore's download site except as noted.

  • AUnit
  • GNATCOLL (except for GNATCOLL-DB, from AdaCore's Github at v21.0.0, with this patch).
    The Python3 binding was built using the Python 3.8.2 distribution from python.org.
  • Gprbuild
  • Langkit
  • Libadalang and tools
  • XMLAda
  • GDB 10.2, as patched here and here (with slight modification).
