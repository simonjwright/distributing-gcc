README for GCC-11.2.0 (arm-eabi)

This is GCC 11.2.0 for arm-eabi built on Mac OS X El Capitan (10.11, Darwin 15).

Also runs on macOS versions up to Big Sur (11, Darwin 20).

Unlike previous GCC packages for macOS, this package installs at a fixed location: /opt/gcc-11.2.0.

Compilers included: Ada, C, C++.

Other software included:

  • binutils-2.36.1
  • newlib-4.1.0
  • GDB 10.2
