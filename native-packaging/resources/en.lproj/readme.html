README for GCC-12.1.0-aarch64

This is GCC 12.1.0-aarch64 built on macOS Monterey (12, Darwin 21) from this source (release notes here), with

  • Command Line Tools 13.3.0
  • Patch for PR104871
  • Python 3.9.11 from python.org

Unlike previous GCC packages for macOS, this package installs at a fixed location: /opt/gcc-12.1.0-aarch64 (though you can move it after installation, if you wish).

Compilers included: Ada, C, C++.

Tools included (all at version 22.0.0 and with the Runtime Library Exception):

  • AUnit from https://github.com/AdaCore/aunit.
  • GNATCOLL from:
    • https://github.com/AdaCore/gnatcoll-core. ZLIB is included, OMP and LZMA are not (the GNATCOLL version for LZMA requires thread support in the system library, not available on macOS).
    • https://github.com/AdaCore/gnatcoll-bindings.
    • https://github.com/AdaCore/gnatcoll-db.
  • Gprbuild from https://github.com/AdaCore/gprbuild.
  • Gprconfig_kb from https://github.com/AdaCore/gprconfig_kb with this patch.
  • Langkit from https://github.com/AdaCore/langkit.
  • Libadalang from https://github.com/AdaCore/libadalang.
  • Libadalang tools from https://github.com/AdaCore/libadalang-tools.
  • XMLAda from https://github.com/AdaCore/xmlada.
